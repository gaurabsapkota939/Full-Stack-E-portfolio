HTML

The Foundation of Every Web Page

Introduction to HTML

HTML (HyperText Markup Language) is the standard markup language for creating webpages and web applications. It structures your content using a system of tags and elements that define the meaning and purpose of each content piece.

Semantic Structure

HTML5 provides semantic elements that clearly define different parts of a webpage, making your code more readable and accessible.

Hypertext Capabilities

Connect documents and resources through hyperlinks, creating the interconnected web of information we use every day.

Universal Compatibility

HTML works across all browsers and devices, ensuring your content is accessible to everyone, everywhere.

Basic Structure

<!DOCTYPE html>
<html lang="en">
  <head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>My First Web Page</title>
  </head>
  <body>
    <header>
      <h1>Hello, World!</h1>
    </header>
    <main>
      <p>This is my first webpage built with HTML.</p>
    </main>
  </body>
</html>

Key Features

  • Uses semantic tags like <header>, <main>, <footer>
  • Supports multimedia (images, video, audio)
  • Forms for user input and interaction
  • Accessibility features with ARIA attributes
  • SEO-friendly structure for better visibility

Did You Know?
